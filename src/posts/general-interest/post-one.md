## Clean Code Matters!

Writing **readable, maintainable code** is as important as making it work. Future-you will thank present-you.

#### **Key Principles**

- Keep functions *small and focused*
- Name variables **clearly**
- Avoid premature optimization
#### **Refactoring Steps**

- Identify code smells
- Write tests *before* changing anything
- Refactor in small, safe increments
- Review with a teammate
**Remember:** code is read far more often than it's written. Treat it like *documentation* that also runs.
